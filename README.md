# Shipment Tracking Logistics Marketplace

## Project Overview

A real-time logistics marketplace platform that enables shippers to post shipment loads and carriers to place bids.

# Shipment Tracking & Logistics Marketplace

A full-stack logistics and shipment tracking platform built using **Spring Boot**, **React.js**, **MySQL**, and **JWT Authentication**.

This project enables users to securely log in, track shipments in real time, view shipment locations on an interactive map, and manage logistics workflows.

---

## Project Features

### Authentication & Security

* JWT-based authentication
* Secure login system
* Protected dashboard routes
* Role-based access control using Spring Security

### Shipment Tracking

* Track shipments using Shipment ID
* Real-time shipment tracking
* Shipment status monitoring
* Latitude and longitude tracking

### Live Map Integration

* Interactive shipment tracking map using **Leaflet.js**
* Displays shipment location using map markers
* Real-time coordinates visualization

### Dashboard

* Professional React dashboard UI
* Shipment details display
* Logout functionality
* Responsive user interface

### API Documentation

* Swagger UI integration for API testing
* Easy API access and testing

## 🚀 How to Run the Project

### Backend Setup

1. Clone repository:

git clone https://github.com/ganeshejdev23/shipment_tracking_logistics_marketplace.git

2. Open project in Eclipse

3. Configure MySQL database

4. Update:

application.properties

with database credentials.

5. Run Spring Boot application

Backend runs on:

http://localhost:8081


### Frontend Setup

Navigate to frontend folder:

cd tracking-frontend

Install dependencies:

npm install

Start React application:

npm start

Frontend runs on:

http://localhost:3000
